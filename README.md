# auto-resume
Leverages rendercv to keep a version controlled resume

## Usage
To preview in VS Code, the following link provides instructions
https://docs.rendercv.com/user_guide/how_to/set_up_vs_code_for_rendercv/


```bash
# to enable live preview
rendercv render --watch Abram_Yorde_CV.yaml
# to rebuild
rendercv render Abram_Yorde_CV.yaml  
```
